# 关于 包活干小程序项目

包活干小程序项目

> 包活干业务平台提供了全流程的线上化展业工具，让服务商可便捷的分配安装人员、跟进订单安装情况，降低服务商日常管理难度，提高管理质量。
该项目的前端模块由本人主导并参与开发。共享平台主要包含一个PC端综合管理系统及4个C端小程序，针对的服务对象有汽车服务商、服务技工，商品供应商，商业运营人员。

# 技术框架
> 微信小程序 + weUi + 小程序UI
