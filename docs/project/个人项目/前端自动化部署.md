# 关于 前端自动化部署

> 前端自动化部署项目，用 js 写一个 CI、CD 工具，主要是为了方便前端开发项目快速部署，使用 node 打造一个自动化部署工具。

# 技术框架
> 服务端：node + koa + koa-static + koa-router + koa-session(鉴权) + socket.io(双向实时通讯)

> 前端：vue3 + elementui + socket.io
