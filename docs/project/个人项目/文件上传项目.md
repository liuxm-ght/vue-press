# 关于 文件上传项目

> Node上传项目，主要是为了方便产品修改、提交、打包、部署文档中心，从而减少对开发的依赖。

# 技术框架
> node + koa + koa-router + simpleGit(用于 git 的操作 add\commit\push 等)

> elememtui
